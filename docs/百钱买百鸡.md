+ 公鸡一只5元，母鸡一只3元，小鸡3只1元，问100元买100只鸡，其中公鸡、母鸡、小鸡各多少只？
<br>基本思路：用变量i、j、k分别表示公鸡、母鸡、小鸡的数量，三者之间应满足关系
1. i+j+k=100;
2. 5*i + 3*j + k/3 = 100;
3. k%3=0

```C
int func() {
  int i, j, k;
  for (i = 0; i < 100; i++) {
    for (j = 0; j < 100; j++) {
      for (k = 0; k < 100; k++) {
        if (i*5+j*3+k/3 == 100 && i+j+k == 100 && k%3 == 0){
	  printf("公鸡%d只,母鸡%d只,小鸡%d只\n", i, j, k);
	}
      }
    }
  }
}
```
